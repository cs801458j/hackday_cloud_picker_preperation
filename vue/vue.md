# Vue.js

## What is vue?
-----------------------
- 프론트엔드 자바스크립트 프레임워크
- Anguler, React에 비해 매우 작고 가벼움. 복잡도도 낮다!(좋은데?)
- 뷰는 CDN으로 사용하기에도 너무나 간편한 것! (원한다면 webpack 사용가능)
- 사용자 인터페이스를 만들기 위한 진보적 framework

## vue 특징
--------------------
- 가상 DOM 사용
- React보다 성능 빠름
- Template 사용
- 서버사이드 렌더링 

### vue 사용기
----------------------
- React의 props와 비슷한 구성이 존재! 
- one-way & two-way binding 지원 

### Directive(지시문)
-----------------------
- Vue 만의 특별한 속성
- Element의 동작을 지시하는 역할 
- "v-"라는 prefix를 가짐
